# 📖 Overview

SummarAI is a modern AI-powered web application that generates concise summaries from long-form text using a fine-tuned **T5 Transformer** model.

The project combines **Natural Language Processing**, **Deep Learning**, and a **FastAPI backend** with a modern SaaS-inspired frontend to deliver fast and accurate text summarization.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AnkurDuttaHQ/SummarAI.git
```

Go to the project directory

```bash
cd SummarAI
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate

Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Locally

```bash
uvicorn app: app --reload
```

Open

```
http://127.0.0.1:8000
```

# 📡 API Endpoint

## POST `/summarize`

### Request

```json
{
    "text":"Artificial Intelligence is transforming industries..."
}
```

### Response

```json
{
    "summary":"AI is transforming industries while improving automation and decision making."
}
```
